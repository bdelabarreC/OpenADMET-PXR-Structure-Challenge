# OpenADMET-PXR-Structure-Challenge

This repository documents a multi-month effort to provide entries for the
OpenADMET PXR structure prediction program.

## Methods and Open-Source Tools Used

The workflow used multiple open-source distributions, including:

- OpenFold3
- Protenix
- Boltz2
- Roshambo
- RDKit
- Additional supporting open-source tools
